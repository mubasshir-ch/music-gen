# Unsupervised Multi-Genre Music Generation

This project explores deep unsupervised generative models for symbolic music, using LSTM Autoencoders, Variational Autoencoders and REMI-tokenized Transformers.

## Links
- **YouTube Presentation:** [https://youtu.be/j3yNocmLYMs](https://youtu.be/j3yNocmLYMs)
- **Project Folder (Report & Assets):** [Google Drive Link](https://drive.google.com/drive/folders/12gIbd5zZU0-J98hfQX5SH9nPcJeWo36R)

## Project Structure
- `task_1/`: LSTM Autoencoder for single-genre music (Done by Shafin Alam).
- `task_2/`: Variational Autoencoder for multi-genre interpolation (Done by Arman Ferdous).
- `task_3/`: Transformer Decoder for long-horizon music generation (Done by Mubasshir Chowdhury).
- `report/`: ICML-style LaTeX report and assets.
